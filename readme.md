**Setting Up**
1. install requirements(there are no dependencies just have python)
2. Activate the virtual environment while in the my_project directory
2. Create a split terminal 
3. Navigate to my_server directory on one terminal and my_client on the other 
4. Run the server.py file frist and when you see server started run client.py file
5. enter a string and you will get string found if file is contained in file and file not found if not
6. to disconnect a client from server enter the string 'bye'
7. to kill the server and start again you need to kill the terminal (ctrl + c)

**testing file**
1. Navigate to the test directory 
2. to test the check string function, use the test_checkstring.py module
3. to test the connection use the test_connection.py
